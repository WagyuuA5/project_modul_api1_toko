## 📝 Deskripsi Proyek
Proyek ini adalah aplikasi e-commerce berbasis mobile yang dirancang dengan prinsip **"Simplicity is the Ultimate Sophistication"**. Fokus utama proyek ini adalah menyediakan pengalaman belanja yang mulus dan sistem manajemen barang yang efisien tanpa komplikasi fitur yang tidak perlu.

Aplikasi ini merupakan bukti kompetensi dalam mengintegrasikan **Flutter Frontend** dengan **REST API** melalui pengujian backend di **Postman**, sebagai syarat kelulusan materi di SMK Telkom Malang.

---

## 🌟 Keunggulan: Mengapa Simpel Lebih Baik?
Berbeda dengan aplikasi toko online pada umumnya yang penuh dengan iklan dan navigasi yang rumit, proyek ini mengedepankan:
* **Efisiensi Performa**: Tanpa animasi berat, aplikasi berjalan sangat ringan bahkan di perangkat spesifikasi rendah.
* **Zero Learning Curve**: Desain yang sangat intuitif membuat pengguna langsung paham cara bertransaksi hanya dalam hitungan detik.
* **Stabilitas Data**: Fokus pada integritas data antara aplikasi dan server melalui integrasi API yang solid.
* **Clean UI**: Menggunakan skema warna modern dan layout yang bersih untuk kenyamanan mata pengguna.

---

## 👥 Fitur Berdasarkan Role

### 🛒 Role: Customer (Pelanggan)
* **Dashboard Modern**: Navigasi kategori produk yang jelas dan menarik.
* **Sistem Keranjang**: Kelola barang yang ingin dibeli dengan mudah.
* **Transaksi Cepat**: Proses checkout yang ringkas dengan pemilihan kurir dan metode pembayaran.
* **History Transaksi**: Pantau semua belanjaan yang pernah dilakukan secara detail.

### ⚙️ Role: Admin (Pengelola)
* **Sistem CRUD Full**: Kendali total untuk Tambah, Lihat, Edit, dan Hapus data produk.
* **Manajemen Gambar**: Fitur unggah foto produk agar tampilan etalase selalu segar.
* **Monitor Stok**: Memastikan informasi barang selalu akurat untuk pelanggan.

---

## 🛠️ Tech Stack & Arsitektur
* **Framework**: Flutter
* **State Management**: Provider (untuk efisiensi logic cart dan produk)
* **Networking**: HTTP/Dio untuk koneksi REST API
* **Backend Testing**: Postman
* **Design Style**: Clean Material Design

## 🚀 Cara Menjalankan
1.  **Clone repo ini**
    ```bash
    git clone https://github.com/WagyuuA5/project_modul_api1_toko.git
    ```
2.  **Masuk ke direktori**
    ```bash
    cd flutter_application_1
    ```
3.  **Ambil package yang diperlukan**
    ```bash
    flutter pub get
    ```
4.  **Jalankan aplikasi**
    ```bash
    flutter run
    ```
